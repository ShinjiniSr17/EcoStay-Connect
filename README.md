# EcoStay Connect 🌿

EcoStay Connect is an AI-powered eco-homestay booking platform that connects travelers with sustainable, community-run accommodations and promotes responsible tourism.

## Features

* Responsive frontend built with Next.js and Tailwind CSS
* Modern landing page with Navbar, Hero section, reusable Cards, and Footer
* Additional routes: About, Dashboard, and Login
* Mobile-friendly and responsive design
* Reusable React component architecture

## Tech Stack

* Next.js (App Router)
* React.js
* Tailwind CSS
* JavaScript
* Git & GitHub

## Getting Started

Install dependencies:

npm install

Run the development server:

npm run dev

Open http://localhost:3000 in your browser to view the application.
